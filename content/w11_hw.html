題目一

n = 10 # 圓形的大小，設置為 10x10
r = 5 # 圓的半徑
center = (5, 5) # 圓心的坐標 (5, 5)

# 使用兩層循環遍歷每個位置
for i in range(n): # 對於每一行
for j in range(n): # 對於每一列
# 計算該點 (i, j) 到圓心 (5, 5) 的距離
distance = ((i - center[0]) ** 2 + (j - center[1]) ** 2) ** 0.5

# 如果距離接近圓的半徑，則打印 * (表示圓形邊界)
if abs(distance - r) < 0.5:
print('*', end=' ')
else:
print(' ', end=' ') # 空白處
print() # 換行

W11題目一(說明)

draw_grid(width, height, grid_pix) 函數

  • 功能：創建整個網格
  • 使用巢狀迴圈遍歷網格的每一個位置
  • 繪製淡灰色的網格線

draw_circle() 函數(數學觀念)

  • distance = math.sqrt((i - center[0])**2 + (j - center[1])**2)
    • 計算網格中每個點到圓心的歐氏距離
    • 使用畢氏定理：√((x1-x2)² + (y1-y2)²)
  • if abs(distance - r) < 0.5
    • 判斷點是否在圓的輪廓附近
    • 誤差範圍設為 0.5，使輪廓看起來更平滑

關鍵變數

  • gs（Grid Size）：每個網格的像素大小
  • tc（Total Count）：網格總數
  • n：圓形所在的網格大小
  • r：圓的半徑
  • center：圓心座標

W11題目二

from browser import html
from browser import document as doc
import math

# 建立畫布
canvas = html.CANVAS(width=400, height=400)
brython_div = doc["brython_div1"]
brython_div <= canvas

# 每格的像素數
gs = 40 # 增加格子大小以便更清楚看到
tc = 10 # 總列數/行數

ctx = canvas.getContext("2d")

def draw_rect(lux, luy, w, h, s=1, c='#cccccc'):
"""繪製矩形框線"""
ctx.lineWidth = s
ctx.strokeStyle = c
ctx.beginPath()
ctx.rect(lux, luy, w, h)
ctx.stroke()

def draw_grid(width, height, grid_pix):
"""繪製網格"""
for i in range(width + 1):
for j in range(height + 1):
draw_rect(i*grid_pix, j*grid_pix, grid_pix, grid_pix, 1, "lightgrey")

def fill_cell(x, y, color):
"""填充特定單元格"""
ctx.fillStyle = color
ctx.fillRect(x*gs, y*gs, gs, gs)

def draw_circle_in_grid(center_x, center_y, radius_cells):
"""
在網格中心繪製圓形
center_x, center_y: 圓心所在的網格座標
radius_cells: 圓的半徑（以格子為單位）
"""
ctx.beginPath()
ctx.fillStyle = "blue"
ctx.arc(
(center_x + 0.5) * gs, # X座標
(center_y + 0.5) * gs, # Y座標
radius_cells * gs, # 半徑
0, # 起始角度
2 * math.pi # 結束角度
)
ctx.fill()

# 繪製網格
draw_grid(tc, tc, gs)

# 在中心繪製圓形
draw_circle_in_grid(tc//2, tc//2, 2)

W11題目二(說明)

引入模組

  • html：用於創建 HTML 元素
  • document：用於訪問網頁的 DOM 元素
  • math：提供數學運算功能

畫布初始化

  • 創建一個 400x400 像素的畫布
  • 將畫布放置在 ID 為 brython_div1 的 div 元素中

繪圖參數

  • gs：Grid Size，每個網格的像素大小
  • tc：Total Count，網格的行列數
  • ctx：Canvas 的 2D 繪圖上下文

draw_rect()

  • 功能：繪製一個帶有指定顏色和線寬的矩形框
  • 預設顏色為灰色，線寬為1像素

draw_grid()

  • 功能：使用巢狀迴圈繪製整個網格
  • 每個網格使用淡灰色線框繪製

fill_cell()

  • 功能：填充指定座標的單一網格
  • 使用指定顏色填滿整個網格

draw_circle_in_grid()

  • 功能：在指定網格中心繪製圓形
  • 使用 arc() 方法繪製圓形
  • 圓心位置為網格中心（加 0.5 是為了置中）
  • 藍色填滿整個圓形

繪圖過程

  • 首先繪製整個網格
  • 然後在網格中心繪製半徑為 2 個格子的藍色圓形

程式變數

  • 可輕鬆調整網格大小 gs
  • 可改變網格數量 tc
  • 可修改圓形位置和大小
  • 可更換顏色和繪圖樣式
